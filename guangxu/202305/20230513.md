## Algorithm

最大线性股票得分

### Description

给定一个整数数组 prices，其中 prices[i] 表示某支股票第 i 天的价格。你只能进行一次买入和一次卖出交易（先买后卖），且必须在买入后卖出。要求在 O (n) 线性时间 内返回能获得的最大利润（即 “最大线性股票得分”）；若无法获得利润，返回 0。​

示例 1：

```
输入：prices = [7,1,5,3,6,4]​
输出：5​
解释：第 2 天买入（价格 1），第 5 天卖出（价格 6），利润 6-1=5。​
```

示例 2：​

```
输入：prices = [7,6,4,3,1]​
输出：0​
解释：价格持续下跌，无盈利空间。​
```

约束条件​
- 1 <= prices.length <= 10^5（需线性时间优化，排除 O (n²) 暴力法）​
- 0 <= prices[i] <= 10^4​


### Solution

```java 
class Solution {
    public int maxLinearStockScore(int[] prices) {
        if (prices.length < 2) {
            return 0;
        }
        int minPrice = prices[0];
        int maxScore = 0;
        for (int i = 1; i < prices.length; i++) {
            if (prices[i] < minPrice) {
                minPrice = prices[i];
            }
            int currentProfit = prices[i] - minPrice;
            if (currentProfit > maxScore) {
                maxScore = currentProfit;
            }
        }
        return maxScore;
    }

    public static void main(String[] args) {
        Solution solution = new Solution();
        System.out.println(solution.maxLinearStockScore(new int[]{7,1,5,3,6,4})); // 5
        System.out.println(solution.maxLinearStockScore(new int[]{7,6,4,3,1}));    // 0
    }
}
```

### Discuss

1. 核心逻辑​

股票单交易最大收益的本质是 “找到最小买入价与后续最大卖出价的差值”，通过一次遍历维护 “历史最低价格” 和 “当前最大利润”，实现线性时间复杂度。​

2. 算法步骤​

初始化变量：​
min_price：记录遍历至今的最低股票价格（初始为 prices[0]）。​
max_score：记录最大利润（初始为 0，无盈利则返回 0）。​

线性遍历：​
对于第 i 天价格 prices[i]，先更新 min_price（取当前 min_price 与 prices[i] 的最小值）。​
计算当前价格与 min_price 的差值（即当天卖出的利润），若大于 max_score 则更新。​

返回结果：遍历结束后 max_score 即为最大线性股票得分。​
